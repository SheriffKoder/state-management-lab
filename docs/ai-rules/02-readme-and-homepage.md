# README and Homepage Rules

Update the root README and app homepage whenever a new tool or iteration is added.

## README responsibilities

Maintain:

- repository purpose
- learning path overview
- available tools
- current progress
- showcase links
- key learnings summary

Update progress checklist:

- [ ] Local State
- [ ] useReducer
- [ ] Context
- [ ] Redux Toolkit
- [ ] Zustand
- [ ] TanStack Query
- [ ] Hybrid Patterns

Add newly completed iterations to the relevant tool section.

---

## Homepage responsibilities

The homepage acts as the visual index.

Always include:

- project title
- short explanation
- links to all learning tools
- links to showcase implementations

Example sections:

Learning
- Local State
- Reducer
- Context
- Redux
- Zustand
- TanStack Query

Showcase
- Redux Task Board
- Zustand Task Board
- TanStack Task Board
- Hybrid Task Board

Homepage must stay updated whenever routes are added.
