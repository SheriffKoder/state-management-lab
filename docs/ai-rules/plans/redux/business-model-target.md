## Business Model Target

### Product Positioning

**ProjectFlow** is a collaborative project operations platform for small service teams and digital agencies.

It helps teams:

- manage multiple client projects
- organize tasks and ownership
- collaborate through comments and activity history
- track project progress
- improve operational visibility

### Primary Users

- project managers
- team leads
- contributors
- workspace admins

### Why this model

This domain naturally exercises:

- relational data
- shared global state
- permissions
- realtime collaboration
- optimistic updates
- complex UI state
- server cache management
