# Iteration Metadata Rules

Every learning iteration must include metadata markdown files.

Required files:

why.md
when-to-use.md
tradeoffs.md
reflection.md

---

## why.md

Explain:

- what problem this solves
- why this tool/feature exists
- what pain it removes

---

## when-to-use.md

Explain:

- ideal scenarios
- scale considerations
- alternatives

---

## tradeoffs.md

Explain:

- complexity introduced
- limitations
- common mistakes
- overengineering risks

---

## reflection.md

Personal learning notes:

- what was learned
- what felt elegant
- what felt painful
- what was surprising
- unresolved questions

Write reflections in first person.
